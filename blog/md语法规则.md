# **Markdown 语言书写指南**

## **1. 标题**
Markdown 使用 `#` 来表示标题，`#` 的数量决定标题的级别（1~6级）。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**效果：**
# 一级标题  
## 二级标题  
### 三级标题  
#### 四级标题  
##### 五级标题  
###### 六级标题  

---

## **2. 段落与换行**
- **段落**：直接输入文字即可形成段落。
- **换行**：在行尾添加两个空格或使用 `<br>` 标签。

```markdown
这是第一行。
这是第二行。  
这是第三行（使用两个空格换行）。
```

**效果：**
这是第一行。  
这是第二行。  
这是第三行（使用两个空格换行）。

---

## **3. 强调**
- **加粗**：用两个星号 `**` 或两个下划线 `__` 包裹文字。
- **斜体**：用一个星号 `*` 或一个下划线 `_` 包裹文字。
- **加粗+斜体**：用三个星号 `***` 或三个下划线 `___` 包裹文字。

```markdown
**加粗文字**  
*斜体文字*  
***加粗且斜体文字***
```

**效果：**
**加粗文字**  
*斜体文字*  
***加粗且斜体文字***

---

## **4. 列表**

### **4.1 无序列表**
使用 `-`、`*` 或 `+` 表示无序列表。

```markdown
- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2
* 项目 3
+ 项目 4
```

**效果：**
- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2
* 项目 3
+ 项目 4

---

### **4.2 有序列表**
使用数字加 `.` 表示有序列表。

```markdown
1. 第一项
2. 第二项
   1. 子项 2.1
   2. 子项 2.2
3. 第三项
```

**效果：**
1. 第一项
2. 第二项
   1. 子项 2.1
   2. 子项 2.2
3. 第三项

---

## **5. 链接与图片**

### **5.1 链接**
使用 `[显示文字](链接地址)` 创建超链接。

```markdown
[访问百度](https://www.baidu.com)
```

**效果：**
[访问百度](https://www.baidu.com)

---

### **5.2 图片**
使用 `![替代文字](图片地址)` 插入图片。

```markdown
![Markdown 图标](https://markdown-here.com/img/icon256.png)
```

**效果：**
![Markdown 图标](https://markdown-here.com/img/icon256.png)

---

## **6. 引用**
使用 `>` 表示引用，可以嵌套多级引用。

```markdown
> 这是一级引用
>> 这是二级引用
>>> 这是三级引用
```

**效果：**
> 这是一级引用  
>> 这是二级引用  
>>> 这是三级引用  

---

## **7. 代码块**

### **7.1 行内代码**
用反引号 `` ` `` 包裹代码。

```markdown
这是 `行内代码` 示例。
```

**效果：**
这是 `行内代码` 示例。

---

### **7.2 多行代码块**
用三个反引号 ````` 包裹代码块，并可以指定语言（如 `python`、`javascript`）。

```markdown
```python
def hello_world():
    print("Hello, world!")
```
```

**效果：**
```python
def hello_world():
    print("Hello, world!")
```

---

## **8. 分割线**
使用三个或以上的 `-`、`*` 或 `_` 创建分割线。

```markdown
---
***
___
```

**效果：**
---
***
___

---

## **9. 表格**
使用 `|` 和 `-` 创建表格，列对齐方式通过 `:` 控制。

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容1  | 内容2    | 内容3  |
| 内容4  | 内容5    | 内容6  |
```

**效果：**

| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容1  | 内容2    | 内容3  |
| 内容4  | 内容5    | 内容6  |

---

## **10. 特殊字符转义**
Markdown 的特殊字符需要使用反斜杠 `\` 转义。

| 特殊字符 | 转义方式 |
|----------|----------|
| `\`      | `\\`     |
| `` ` ``  | `` \` `` |
| `*`      | `\*`     |
| `_`      | `\_`     |
| `{}`     | `\{\}`   |
| `[]`     | `\[\]`   |
| `()`     | `\(\)`   |
| `#`      | `\#`     |
| `+`      | `\+`     |
| `-`      | `\-`     |
| `.`      | `\.`     |
| `!`      | `\!`     |

示例：
```markdown
\*这是加粗的星号符号\*
```

**效果：**
\*这是加粗的星号符号\*

---

## **11. 任务列表**
使用 `- [ ]` 表示未完成任务，`- [x]` 表示已完成任务。

```markdown
- [x] 已完成任务
- [ ] 未完成任务
```

**效果：**
- [x] 已完成任务
- [ ] 未完成任务

---

## **12. HTML 支持**
Markdown 支持直接嵌入 HTML 标签。

```markdown
这是一个 <b>加粗</b> 的 HTML 示例。
```

**效果：**
这是一个 <b>加粗</b> 的 HTML 示例。

---

## **13. 注脚**
使用 `[^注脚标识]` 添加注脚。

```markdown
这是一个带有注脚的句子[^1]。

[^1]: 这是注脚的内容。
```

**效果：**
这是一个带有注脚的句子[^1]。

[^1]: 这是注脚的内容。

---

## **14. 嵌套组合**
Markdown 的各种语法可以组合使用，比如引用中嵌套列表、代码块等。

```markdown
> 这是引用
> - 列表项 1
> - 列表项 2
>   ```javascript
>   console.log("代码块示例");
>   ```
```

**效果：**
> 这是引用
> - 列表项 1
> - 列表项 2
>   ```javascript
>   console.log("代码块示例");
>   ```

## **15. 常用快捷键（编辑器支持）**
- **加粗**：`Ctrl + B` 或 `Cmd + B`
- **斜体**：`Ctrl + I` 或 `Cmd + I`
- **插入链接**：`Ctrl + K` 或 `Cmd + K`
- **插入代码块**：````
- **插入图片**：`Ctrl + Shift + I` 或 `Cmd + Shift + I`
