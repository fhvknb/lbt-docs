Whisper дҪҝз”Ёе‘Ҫд»ӨиҜҙжҳҺ

MacOS дёҠдҪҝз”Ё OpenAI Whisper зҡ„иҜҰз»Ҷж•ҷзЁӢвҖӢ

1. зҺҜеўғеҮҶеӨҮвҖӢ

1.1 е®үиЈ… HomebrewвҖӢ

Homebrew жҳҜ macOS еёёз”Ёзҡ„иҪҜд»¶еҢ…з®ЎзҗҶеҷЁгҖӮеҰӮжһңе°ҡжңӘе®үиЈ…пјҢеҸҜд»ҘйҖҡиҝҮд»ҘдёӢе‘Ҫд»Өе®үиЈ…пјҡ

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

е®үиЈ…е®ҢжҲҗеҗҺпјҢиҝҗиЎҢд»ҘдёӢе‘Ҫд»ӨзЎ®дҝқ Homebrew жӯЈеёёе·ҘдҪңпјҡ

brew update
brew doctor

1.2 е®үиЈ… PythonвҖӢ

Whisper йңҖиҰҒ Python зҺҜеўғпјҢе»әи®®дҪҝз”Ё Python 3.8 жҲ–жӣҙй«ҳзүҲжң¬гҖӮеҸҜд»ҘйҖҡиҝҮ Homebrew е®үиЈ…пјҡ

brew install python

е®үиЈ…е®ҢжҲҗеҗҺпјҢжЈҖжҹҘ Python зүҲжң¬пјҡ

python3 --version

еҰӮжһңзүҲжң¬еҸ·жҳҜ 3.8 жҲ–жӣҙй«ҳпјҢеҲҷеҸҜд»Ҙз»§з»ӯгҖӮ

2. е®үиЈ… WhisperвҖӢ

2.1 е®үиЈ… pip е’Ң virtualenvвҖӢ

зЎ®дҝқ pip е·Іе®үиЈ…пјҢе№¶е®үиЈ… virtualenv жқҘеҲӣе»әйҡ”зҰ»зҡ„ Python зҺҜеўғпјҡ

python3 -m ensurepip --upgrade
pip3 install virtualenv

2.2 еҲӣе»әиҷҡжӢҹзҺҜеўғвҖӢ

еңЁйЎ№зӣ®ж–Үд»¶еӨ№дёӯеҲӣе»әдёҖдёӘиҷҡжӢҹзҺҜеўғпјҡ

mkdir whisper_project
cd whisper_project
python3 -m virtualenv venv
source venv/bin/activate

жҝҖжҙ»иҷҡжӢҹзҺҜеўғеҗҺпјҢдҪ дјҡзңӢеҲ°е‘Ҫд»ӨиЎҢеүҚйқўжңү (venv) зҡ„ж Үи®°гҖӮ

2.3 е®үиЈ…дҫқиө–вҖӢ

Whisper дҫқиө– PyTorch е’Ң ffmpegгҖӮдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өе®үиЈ…пјҡ

pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
pip install git+https://github.com/openai/whisper.git

еҰӮжһңдҪ зҡ„зі»з»ҹж”ҜжҢҒ GPUпјҢеҸҜд»Ҙе®үиЈ… GPU зүҲжң¬зҡ„ PyTorchпјҢе…·дҪ“е®үиЈ…ж–№жі•еҸҜд»ҘеҸӮиҖғ PyTorch е®ҳзҪ‘гҖӮ

3. е®үиЈ… ffmpegвҖӢ

Whisper йңҖиҰҒ ffmpeg жқҘеӨ„зҗҶйҹійў‘ж–Үд»¶гҖӮеҸҜд»ҘйҖҡиҝҮ Homebrew е®үиЈ…пјҡ

brew install ffmpeg

е®үиЈ…е®ҢжҲҗеҗҺпјҢйӘҢиҜҒ ffmpeg жҳҜеҗҰжӯЈеёёе·ҘдҪңпјҡ

ffmpeg -version

4. дҪҝз”Ё WhisperвҖӢ

4.1 дёӢиҪҪйҹійў‘ж–Үд»¶вҖӢ

еҮҶеӨҮеҘҪйңҖиҰҒиҪ¬жҚўзҡ„йҹійў‘ж–Үд»¶пјҢзЎ®дҝқж–Үд»¶ж јејҸдёә .mp3, .wav жҲ–е…¶д»–еёёи§Ғйҹійў‘ж јејҸгҖӮ

4.2 иҪ¬жҚўйҹійў‘дёәж–Үжң¬вҖӢ

иҝҗиЎҢд»ҘдёӢе‘Ҫд»ӨдҪҝз”Ё Whisper е°Ҷйҹійў‘иҪ¬жҚўдёәж–Үжң¬пјҡ

whisper example.mp3 --model small

е…¶дёӯпјҡ

  • example.mp3 жҳҜйҹійў‘ж–Үд»¶зҡ„и·Ҝеҫ„гҖӮ
  • --model жҢҮе®ҡдҪҝз”Ёзҡ„жЁЎеһӢеӨ§е°ҸпјҲеҸҜйҖүеҖјпјҡtiny, base, small, medium, largeпјүгҖӮжЁЎеһӢи¶ҠеӨ§пјҢзІҫеәҰи¶Ҡй«ҳпјҢдҪҶиҝҗиЎҢйҖҹеәҰи¶Ҡж…ўгҖӮ

4.3 жҢҮе®ҡиҜӯиЁҖвҖӢ

еҰӮжһңйҹійў‘ж–Үд»¶жҳҜйқһиӢұиҜӯиҜӯиЁҖпјҢеҸҜд»ҘйҖҡиҝҮ --language еҸӮж•°жҢҮе®ҡиҜӯиЁҖгҖӮдҫӢеҰӮпјҡ

whisper example.mp3 --model small --language zh

5. й«ҳзә§дҪҝз”ЁвҖӢ

5.1 иҪ¬жҚўдёәеӯ—е№•ж–Үд»¶вҖӢ

еҸҜд»Ҙз”ҹжҲҗеӯ—е№•ж–Үд»¶пјҲеҰӮ .srt жҲ– .vttпјүпјҡ

whisper example.mp3 --model small --output_format srt

5.2 жү№йҮҸеӨ„зҗҶвҖӢ

еҰӮжһңжңүеӨҡдёӘйҹійў‘ж–Үд»¶пјҢеҸҜд»ҘйҖҡиҝҮжү№йҮҸеӨ„зҗҶи„ҡжң¬жқҘиҪ¬жҚўпјҡ

for file in *.mp3; do
    whisper "$file" --model small;
done

6. жҖ§иғҪдјҳеҢ–вҖӢ

еҰӮжһңдҪ зҡ„зі»з»ҹж”ҜжҢҒ GPUпјҢеҸҜд»Ҙе®үиЈ… GPU зүҲжң¬зҡ„ PyTorchпјҢе№¶дҪҝз”Ёд»ҘдёӢе‘Ҫд»ӨеҗҜз”Ё GPU еҠ йҖҹпјҡ

pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

иҝҗиЎҢ Whisper ж—¶пјҢе®ғдјҡиҮӘеҠЁжЈҖжөӢе№¶дҪҝз”Ё GPUгҖӮ

7. еёёи§Ғй—®йўҳвҖӢ

7.1 ffmpeg жңӘе®үиЈ…жҲ–жңӘжүҫеҲ°вҖӢ

зЎ®дҝқ ffmpeg е·Іе®үиЈ…пјҢе№¶дё”и·Ҝеҫ„жӯЈзЎ®гҖӮеҰӮжһңд»Қз„¶жҠҘй”ҷпјҢеҸҜд»Ҙе°қиҜ•йҮҚж–°е®үиЈ…пјҡ

brew reinstall ffmpeg

7.2 еҶ…еӯҳдёҚи¶івҖӢ

еҰӮжһңжЁЎеһӢиҝҮеӨ§пјҢеҸҜиғҪдјҡеҜјиҮҙеҶ…еӯҳдёҚи¶ігҖӮеҸҜд»Ҙе°қиҜ•дҪҝз”Ёиҫғе°Ҹзҡ„жЁЎеһӢпјҲеҰӮ tiny жҲ– baseпјүгҖӮ

иҝҷж ·пјҢдҪ е°ұеҸҜд»ҘеңЁ macOS дёҠйЎәеҲ©дҪҝз”Ё Whisper иҝӣиЎҢиҜӯйҹіиҜҶеҲ«дәҶпјҒеҰӮжһңжңүе…¶д»–й—®йўҳпјҢиҜ·йҡҸж—¶иҜўй—®гҖӮ
