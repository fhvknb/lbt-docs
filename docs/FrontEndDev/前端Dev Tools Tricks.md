# Dev Tools Tricks

1. `console.log` 添加 style

`console.log('%c hello world!', 'font-size: 20px; color: red;')`

2. `console.assert` 测试断言
3. `console.clear` 
4. `console.dir(dom)` 
5. `console.groupCollapsed(key)` ＆　`console.groupEnd(key)` 对打印的内容进行分组，方便查看 
6. `console.count('key')` 
7. `console.time(begin key)` & `console.timeEnd('begin key')` 
8. `console.table('map data')`
