中国A股与港股交易规则详解

中国A股与港股交易规则详解​

一、中国A股交易规则​

中国股市（A股市场）的交收时间遵循T+1结算制度，具体如下：

  1. 交收时间和规则：

    • T+1结算制度：
      • 股票交易采用T+1制度：
        • 当天买入的股票只能在下一个交易日才能卖出。
      • 资金结算则是实时完成：
        • 卖出股票后，资金可以在当天使用（例如买入其他股票或进行其他投资）。
    • 交收时间：
      • 股票交易的交收时间为**交易日后的第二个工作日（T+1）**完成。
      • 具体流程：
        • T日（交易日）：投资者完成买卖交易。
        • T+1日：交易所和证券登记机构完成股票和资金的交收。
    • 资金使用：
      • 卖出股票后的资金可以在当天使用，但资金的正式交收是在T+1日完成。

  2. 交易时间：

    • 中国股市的交易时间为：
      • 上午：09:30 - 11:30
      • 下午：13:00 - 15:00

  3. 其他注意事项：

    • 休市时间：
      • 中国股市在法定节假日、周末休市，交收时间顺延。
    • 资金流转：
      • 卖出股票后资金可以用于当天的其他交易，但无法直接提现，提现需等到交收完成。

二、港股交易规则​

港股（香港股票市场）的交易规则和交易时间有其独特的特点，具体如下：

  1. 交易时间：

    • 竞价时段（开市前时段）：
      • 时间：09:00 - 09:30
      • 细分为：
        • 09:00 - 09:15：输入买卖盘时段，投资者可以输入、修改或取消买卖盘指令。
        • 09:15 - 09:20：不可取消时段，投资者只能输入新买卖盘，无法修改或取消。
        • 09:20 - 09:28：随机收市时段，系统会在这一时间内随机结束竞价。
      • 功能：通过集合竞价确定开盘价。
    • 持续交易时段（正常交易时段）：
      • 上午时段：09:30 - 12:00
      • 午间休市：12:00 - 13:00
      • 下午时段：13:00 - 16:00
    • 收市竞价时段：
      • 时间：16:00 - 16:10
      • 细分为：
        • 16:00 - 16:06：输入买卖盘时段，投资者可以输入、修改或取消买卖盘。
        • 16:06 - 16:08：不可取消时段，投资者只能输入新买卖盘，无法修改或取消。
        • 16:08 - 16:10：随机收市时段，系统在这一时间内随机结束竞价。
      • 功能：通过集合竞价确定收盘价。

  2. 交易规则：

    • 交易单位：
      • 港股的交易以“手”为单位，每手的股数由上市公司自行决定，通常为100股或500股，部分股票可能是1000股或更多。
    • 价格变动单位（最小报价单位）：
      • 港股的价格变动单位与股票价格相关，价格越高，最小变动单位越大。例如：
        • 价格 ≤ 0.25港元，最小变动单位为 0.001港元。
        • 价格 > 20港元，最小变动单位为 0.1港元。
      • 具体的价格变动单位可以参考港交所的规则。
    • T+2结算制度：
      • 港股采用T+2结算制度，即交易日后的第二个工作日完成资金和股票的交收。
    • T+0交易制度： 当天买入的股票当天可以卖出
    • 无涨跌幅限制：
      • 港股市场没有像A股那样的涨跌幅限制，但为了防止异常波动，港股引入了市场波动调节机制（即熔断机制），适用于恒生指数成分股和部分活跃股票：
        • 当股票价格较参考价格波动超过±5%时，会触发机制，将交易限制在±5%的范围内，持续5分钟。
    • 买卖盘撮合：
      • 港股采用价格优先、时间优先的撮合原则：
        • 价格优先：买入价高的优先，卖出价低的优先。
        • 时间优先：在同一价格下，较早提交的订单优先。
    • 交易费用：
      • 投资港股需要支付以下费用：
        • 佣金：由券商收取，通常为交易金额的0.1%-0.25%。
        • 香港交易所费用：包括交易费（0.005%）和交易系统使用费（每笔0.5港元）。
        • 印花税：交易金额的0.13%（四舍五入到最接近的1港元）。
        • 过户费：每手2港元，由香港中央结算公司收取。

  3. 其他注意事项：

    • 港股通：
      • 通过沪港通或深港通参与港股交易的投资者需要遵守额外的规则，例如港股通标的范围、额度限制等。
    • 交易货币：
      • 港股主要以**港元（HKD）**交易，但部分股票也支持人民币（CNY）或美元（USD）交易。
    • 股份类别：
      • 港股有不同的股份类别，例如普通股（-H）、生物科技股（-B）等，投资者需注意其特点和风险。

三、A股与港股比较​

中国股市采用T+1制度，而港股市场采用T+2制度。主要区别在于：

  • A股（T+1）：
    • 当天买入的股票不能当天卖出，资金结算相对快。
  • 港股（T+2）：
    • 交收时间更长，股票和资金的交收需要两个工作日。
  • 其他差异：
    • A股有明确的涨跌幅限制，港股则无涨跌幅限制但引入熔断机制。
    • 交易时间方面，A股为连续交易时段，港股则有竞价时段和收市竞价时段。
