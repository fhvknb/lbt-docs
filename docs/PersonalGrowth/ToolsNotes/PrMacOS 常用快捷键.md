# Pr　MacOS 常用快捷键

以下是Mac版Adobe Premiere Pro常用剪辑的快捷键说明，帮助你提高剪辑效率：

### 常用快捷键

#### 基本操作
- **新建项目**: `Command + N`
- **打开项目**: `Command + O`
- **保存项目**: `Command + S`
- **导出媒体**: `Command + M`

#### 时间线操作
- **播放/暂停**: `Space`
- **停止**: `K`
- **向前/向后播放**: `L` / `J`
- **向前/向后跳过一帧**: `→` / `←`
- **向前/向后跳过十帧**: `Shift + →` / `Shift + ←`
- **切割剪辑**: `Command + K`
- **选择所有剪辑**: `Command + A`
- **删除剪辑**: `Delete`

#### 剪辑控制
- **放大时间线**: `Command + =`
- **缩小时间线**: `Command + -`
- **切换选择工具和手形工具**: `V`（选择工具） / `H`（手形工具）

#### 标签和标记
- **添加标记**: `M`
- **跳转到下一个标记**: `Shift + M`
- **跳转到上一个标记**: `Command + Shift + M`

#### 其他实用快捷键
- **撤销**: `Command + Z`
- **重做**: `Command + Shift + Z`
- **全屏播放**: `Control + Command + F`

### 提示
- 可以通过 `Command + Option + K` 打开快捷键自定义设置，查看或修改快捷键。
- 熟悉这些快捷键可以显著提高你的剪辑速度和工作效率。
