Photoshop中快速将白色背景变透明的详细步骤

Photoshop中快速将白色背景变透明的详细步骤​

步骤1：打开图片​

  1. 打开 Photoshop。
  2. 导入需要处理的图片。

步骤2：解锁图层​

  1. 在 图层面板 中，找到图片所在的图层（通常是“背景”图层）。
  2. 如果图层被锁定，双击图层名称，弹出“新建图层”对话框，点击 确定 解锁图层。

步骤3：选择白色背景​

  • 快速选择工具方法：

    1. 点击左侧工具栏中的 魔棒工具（快捷键：W）。
    2. 在顶部工具栏中，将“容差”设置为 20-30（根据背景白色的纯度调整）。
    3. 点击白色背景区域，魔棒工具会自动选中背景。
    4. 如果有未选中的部分，可以按住 Shift 键继续点击添加选区。

  • 选择颜色范围方法（更精准）：

    1. 点击顶部菜单栏的 选择 > 色彩范围。
    2. 在弹出的窗口中，使用吸管工具点击白色背景。
    3. 调整“容差”滑块，直到选中所有白色区域。
    4. 点击 确定。

步骤4：删除背景​

  1. 确保白色背景被选中后，按键盘上的 Delete 键删除选区内容。
  2. 如果选中了错误的区域，可以按 Ctrl+Shift+I（Windows）或 Cmd+Shift+I（Mac）反选，再删除。

步骤5：保存为透明格式​

  1. 删除背景后，图片会显示为灰白相间的格子，这表示背景已经透明。
  2. 点击 文件 > 导出 > 导出为。
  3. 在导出设置中选择 PNG 格式（支持透明背景）。
  4. 点击 导出 并保存文件。

小提示​

  • 如果图片边缘有白色残留，可以使用 选择 > 修改 > 收缩 或 羽化 工具优化选区。
  • 使用 橡皮擦工具 或 蒙版 手动修整边缘区域。

这样，你就可以快速将白色背景变成透明了！
