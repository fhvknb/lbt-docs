# foobar 设置参数说明

## 音频效果处理
1. Echo（回声）
- 功能：模拟声音在房间、山谷等环境中反射的效果。
- 作用：为声音增加空间感和深度，听起来像在一个大房间里唱歌。
2. Equalizer（均衡器）
- 功能：调整音频中不同频段（如低音、中音、高音）的音量。
- 作用：可以根据喜好或不同音乐风格（如摇滚、古典）来“调音”，让低音更震撼，或人声更清晰。
3. Reverb（混响）
- 功能：比回声更复杂，模拟声音在封闭空间内多次
- 反射、融合的效果。
- 作用：让声音听起来更丰满、更有“空气感”，常用于美化人声或乐器声
4. Push The Tempo（加速播放）
- 功能：加快音乐的播放速度。
- 作用：可以让一首慢歌变快，适合跑步、做家务时听，或者单纯想快速听完一首歌。


## 音量与动态控制
1. Advanced Limiter（高级限制器)
- 功能：自动检测并压缩音频中过大的音量峰值。
- 作用：防止音量突然过大导致爆音或损伤听力，让整体音量更平稳。
2. Hard -6dB limiter（硬限幅器-6dB)
- 功能:将所有音频信号的音量强制限制在-6dB以下。
- 作用：比高级限制器更粗暴，确保音量不会超过设定值，常用于防止设备过载。
3. Skip Silence（跳过静音）
- 功能：智能检测音频中的长时间静音片段。
- 作用：自动跳过这些静音部分，直接播放有声音的内容，节省时间。

## 声道与采样率处理
1. Convert mono to stereo（单声道转立体声）
- 功能：将原本只有一个声道的音频（如老式收音机）转换为两个声道。
- 作用：让单声道音频听起来更有空间感，但无法创造出真正的立体声效果。
2. Crossfader（交叉淡化器）
- 功能：在两首歌之间切换时，实现平滑的淡入淡出过渡。
- 作用：避免两首歌切换时出现突兀的停顿，让播放更流畅。
2. Downmix channels to mono/stereo（声道下混为单声道/立体声)
- 功能：将多声道音频（如5.1环绕声）合并为单声道或立体声。
- 作用：当你的设备不支持多声道输出（如普通耳机）时，确保能听到所有声音。
3. Meier Crossfeed(Meier交叉馈送)
- 功能：模拟人耳在听立体声时，左右耳会听到对方声道声音的自然现象。
- 作用：用耳机听音乐时，能减少“头中效应”，音感觉更像是从前方传来，而不是在脑子里。
4. Resampler（重采样器）
- 功能：改变音频的采样率。
- 作用：将低采样率的音频转换为高采样率，以匹配播放器或输出设备的要求，理论上能提升音质。
5. Reverse stereo channels（反转立体声通道）
- 功能：交换左右声道的音频信号。
