

### 研发相关系统
**PLM**: Product Lifecycle Management  （产品生命周期）
**ALM**: Application Lifecycle Management 　（软件生命周期）
**SOP**: Start Of Product（开始量产）
**SOR** : **Statement of Requirements** 的缩写，意为**需求说明书**或**需求陈述**。
**IPD**: 集成产品开发（Integrated Product Development）

### 车辆相关
**AE**: Automotive Engineering (车辆工程)
**FSD**: Full Self Driving (全智能驾驶)
**HMI** :  Human Machine Interface(人机交互接口)

**IDCU**:   Intelligent Cabin  Control Unit  智能座舱域控制器
**ADCU**:  **Autonomous Driving Control Unit**‌（自动驾驶域控制器）

**VCU: （Vehicle Control Unit）**  **整车控制单元**  
**VIU**:  **Vehicle Interface Unit** 的缩写，意为**车辆接口单元**。

**ECU** 是 **Electronic Control Unit** 的缩写，意为**电子控制单元**
- **发动机控制单元（Engine Control Unit）**
- **变速器控制单元（Transmission Control Unit, TCU）**
- **车身控制单元（Body Control Module, BCM）**
- **底盘控制单元（Chassis Control Unit）**

**MCU** 是 **Motor Control Unit** 的缩写，意为**电机控制单元**。
**CDC** 是 **Continuous Damping Control** 的缩写，意为**连续可变阻尼控制系统**。
**EPS** 是 **Electric Power Steering** 的缩写，意为**电动助力转向系统**。

**BMS**:  **Battery Management System**　**电池管理系统**
- SOC（State of Charge，荷电状态）：估算电池剩余电量。
- SOH（State of Health，健康状态）：评估电池的健康状况，预测电池寿命。

**ECO** : 驾驶模式之一**经济模式**（Economy Mode）



### 安全领域相关
**FSC**: Fail Safe Control　（失效安全控制）  
**IDPS**（Intrusion Detection and Prevention System，入侵检测与防御系统） 
**VSOC** :  **Virtual Security Operations Center**  **虚拟安全运营中心**   
**UDS** :  **Unified Diagnostic Services** 的缩写，意为**统一诊断服务**

| 缩写       | 全称                                        | 中文含义       | 主要作用                      |
| -------- | ----------------------------------------- | ---------- | ------------------------- |
| **AVAS** | Acoustic Vehicle Alerting System          | 声学车辆警报系统   | 为低噪声的电动车行驶时提供警示声音，保障行人安全。 |
| **RCSW** | Rear Cross Side Warning                   | 后方横向来车警告   | 倒车时监测后方横向来车情况，防止碰撞。       |
| **LCSW** | Lane Change Side Warning                  | 变道侧向警告系统   | 监测侧后方盲区，避免变道时发生碰撞。        |
| **IC**   | Instrument Cluster                        | 仪表盘        | 显示车辆的各种信息，如速度、电量、故障灯。     |
| **FCM**  | Forward Collision Mitigation              | 前向碰撞缓解系统   | 提前警告并主动制动以减轻碰撞风险。         |
| **FRM**  | Front Radar Module                        | 前雷达模块      | 探测前方障碍物，支持辅助驾驶功能。         |
| **ESC**  | Electronic Stability Control              | 电子稳定控制系统   | 防止车辆失控，保持行驶稳定性。           |
| **ABSI** | Anti-lock Braking System with Integration | 集成式防抱死制动系统 | 防止紧急刹车时车轮抱死，提升安全性和稳定性。    |
| **EHB**  | Electro-Hydraulic Braking                 | 电子液压制动系统   | 提供精确的制动力分配，支持能量回收。        |
| **EPS**  | Electric Power Steering                   | 电动助力转向系统   | 提供方向盘助力，提升操控和舒适性。         |
| **MCU**  | Motor Control Unit                        | 电机控制单元     | 控制电机的运行，包括转速、扭矩和方向。       |
| **HCM**  | Hybrid Control Module                     | 混合动力控制模块   | 协调发动机与电机的工作，优化混动系统性能。     |
| **DCDC** | DC-DC Converter                           | 直流-直流变换器   | 将高压直流电转换为低压直流电，为低压设备供电。   |
| **OBC**  | On-Board Charger                          | 车载充电机      | 将外部交流电转换为直流电，为电池充电。       |
| **EAS**  | Electric Air Suspension                   | 电子空气悬架系统   | 通过空气悬架调节车辆高度和舒适性。         |
| **WPTC** | Water-cooled Plate Temperature Control    | 水冷板温控系统    | 冷却电池、电机等部件，优化热管理。         |
| **BMS**  | Battery Management System                 | 电池管理系统     | 管理电池状态，确保电池安全高效运行。        |
| **APU**  | Auxiliary Power Unit                      | 辅助动力单元     | 提供辅助动力或电力，常用于增程器或飞机中。     |
| **GCU**  | Gearbox Control Unit                      | 变速箱控制单元    | 控制变速箱的换挡逻辑和动力传输。          |
| **ECU**  | Electronic Control Unit                   | 电子控制单元     | 控制车辆的特定功能模块（发动机、变速器等）。    |
| **OHAC** | Optimized Heating, Air Conditioning       | 优化的加热与空调系统 | 提供智能化的空调和暖风控制，提升车内舒适性。    |
